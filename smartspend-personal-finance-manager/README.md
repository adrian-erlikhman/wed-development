# SmartSpend: Personal Finance Manager

## Project Description
SmartSpend helps users track budgets, categorize expenses, visualize spending trends, and set financial goals.

## Technologies and Tools Used
- React
- Firebase
- Chart.js
- Plaid API (or mock data)

## Setup & Installation
1. **Prerequisites**
   - Node.js
   - Firebase account
2. **Clone the repository**
   ```bash
   git clone <repo-url>
   cd smartspend-personal-finance-manager
   ```
3. **Install dependencies**
   ```bash
   npm install
   ```
4. **Environment Variables**
   Create a `.env` file with Firebase config and Plaid keys.
5. **Run the app**
   ```bash
   npm start
   ```
   Visit `http://localhost:3000` in your browser.
6. **Build for production**
   ```bash
   npm run build
   ```
7. **Deployment**
   Deploy using Firebase Hosting, Netlify, or similar platforms.
